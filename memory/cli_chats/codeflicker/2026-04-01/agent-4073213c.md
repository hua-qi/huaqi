## 3. ihr-cards 引用配置

### 卡片库名称: **`ihr_widget_provider`**

**从后端获取的配置格式** (API: `/magic-ai/api/config/getCardConfig`):

```typescript
export interface WidgetProvider {
  name?: string;           // 例如: "ihr_widget_provider"
  entry?: string;          // 例如: "https://localhost:3001/remoteEntry.js"
  widgetList?: string[];   // 包含的 widget 列表，如 ["CardA", "CardB"]
}
```

### 联调支持

**URL 参数方式**:
```
http://localhost:3000?localCards=http://localhost:3001/remoteEntry.js
```

这会将 `ihr_widget_provider` 的 entry 替换为本地地址，支持在开发时测试本地卡片库。
