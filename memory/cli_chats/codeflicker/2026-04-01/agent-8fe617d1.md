[2026-04-01 20:25:21] [assistant]: ## codeflicker compact 命令详解

### 功能与作用

`compact` 命令会对当前会话的历史对话进行**压缩处理**，同时在会话中保留一个**摘要信息**。

**核心目的**：减少上下文窗口中的 token 体积，但仍保留核心线索，便于继续对话。

---

### 适用场景

当会话已经较长、接近上下文容量上限时，使用 `/compact` 可：
- 释放容量
- 保留摘要确保后续讨论不丢失关键信息

**推荐时机**：当状态栏显示容量低于 30% 时，推荐执行 `/compact` 来压缩历史。

---

### 行为方式

1. 将较早的对话内容压缩成一个**摘要**
2. 将摘要信息保留在会话中
3. 后续对话仍能基于该摘要继续进行

---

### 使用方法

在当前会话中直接输入：
```
/compact
```

---

### 注意事项

1. **细节丢失**：压缩会丢失部分细节，依赖摘要来维持上下文
2. **提前导出**：如果需要保留全部细节，请在压缩前使用 `/export` 导出会话，或使用 `/branch` 分支对话以备查阅
3. **重新查看**：历史被压缩后，如需重新查看详细历史，可通过 `/export` 导出会话上下文

---

### 相关命令

| 命令 | 用途 |
|------|------|
| `/compact` | 压缩历史对话为摘要 |
| `/export` | 导出会话上下文 |
| `/branch` | 分支对话 |
| 状态栏 | 显示当前容量使用情况 |
