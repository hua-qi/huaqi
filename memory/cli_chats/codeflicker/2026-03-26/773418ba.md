[2026-03-27 01:26:42] [user]: https://github.com/eosphoros-ai/DB-GPT 这个项目是做什么的
[2026-03-27 01:27:15] [assistant]: DB-GPT 是一个开源的智能代理 AI 数据助手（Agentic AI Data Assistant），面向下一代 AI + 数据产品。

主要功能：
- 通过自然语言与数据库交互
- AI 驱动的数据查询、分析和管理
- 支持多种数据库和 LLM 模型
- 提供数据智能代理框架

该项目在 GitHub 上已有 18.4k+ Star，是数据+AI 领域较活跃的开源项目。
[2026-03-27 01:32:19] [user]: DB-GPT 是不是有经历过 workflow 的阶段
[2026-03-27 01:33:46] [assistant]: 是的，DB-GPT 经历了从固定场景到 workflow/Agent 架构的演进：

**演进历程：**

1. **早期（v0.4.x 之前）**：6个固定场景
   - ChatData、ChatExcel、ChatDB、ChatKnowledge、ChatAgents、ChatDashboard

2. **v0.5.0（Workflow 阶段）**：
   - 引入 **AWEL** (Agentic Workflow Expression Language)
   - 用户可通过可视化 workflow 编排构建数据应用
   - 原固定场景逐步重写成 Data Apps

3. **v0.6.0（AWEL 2.0）**：
   - 支持更复杂的编排
   - 多种应用构建模式：multi-agent 自动规划、任务流编排、单 agent、原生应用

4. **v0.7.0（当前）**：
   - 支持 MCP 协议
   - 模块架构重组
   - 配置系统重构为 `.toml` 格式

**核心理念转变：** 从"提供固定的数据对话场景" 演进为 "通过 AWEL workflow 和 agents 构建 AI-native 数据应用" 的开发框架。
