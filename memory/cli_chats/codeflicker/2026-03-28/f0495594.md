**2. Missing command 根因**

这些命令组都缺少 `@xxx_app.callback(invoke_without_command=True)` 装饰器。

Typer 的处理逻辑：直接运行命令组（不带子命令）时，会找 callback；有 callback 则执行（如 `config` 就有，所以能显示帮助）；没有 callback 就报 `Missing command`。

`status` 是直接挂在主 app 上的命令，不是子应用，所以不受此影响。

**修复方式**：为 `profile_app`、`personality_app`、`pipeline_app`、`system_app` 各加一个 callback：
```python
@profile_app.callback(invoke_without_command=True)
def profile_callback(ctx: typer.Context):
    if ctx.invoked_subcommand is None:
        ctx.get_help()
```
