```typescript
import React, { useState } from 'react';
import { AppRenderer } from '@mcp-ui/client';
import type { McpUiResourcePart } from '../types';

interface Props {
  resourceUri: string;
  uiMetadata?: McpUiResourcePart['uiMetadata'];
  onMessage?: (text: string) => void;
  onLayout?: () => void;
}

export const CardMessage: React.FC<Props> = ({ resourceUri, uiMetadata, onMessage, onLayout }) => {
  const { width = 560 } = uiMetadata?.['preferred-frame-size'] ?? {};
  const sandboxUrl = new URL('/sandbox_proxy.html', window.location.href);
  const [iframeHeight, setIframeHeight] = useState<number | undefined>(undefined);

  return (
    <div style={{
      border: iframeHeight ? '1px solid #e0e0e0' : 'none',
      borderRadius: 8,
      overflow: 'hidden',
      width,
      height: iframeHeight ?? 0,
      transition: 'height 0.2s ease',
    }}>
      <AppRenderer
        toolName="query_employee_trend"
        sandbox={{ url: sandboxUrl }}
        toolResourceUri={resourceUri}
        onReadResource={async ({ uri }) => {
          const res = await fetch(`/resource-proxy?uri=${encodeURIComponent(uri)}`);
          return res.json();
        }}
        onSizeChanged={({ height }) => {
          if (height !== undefined) {
            setIframeHeight(height);
            onLayout?.();
          }
        }}
        onMessage={async (params) => {
          const textBlock = params.content.find((c: any) => c.type === 'text');
          if (textBlock && onMessage) {
            onMessage((textBlock as any).text);
          }
          return {};
        }}
        onError={(e) => console.error('[AppRenderer]', e)}
      />
    </div>
  );
};
```

**关键点：**
- `onReadResource` 回调通过 `/resource-proxy?uri=...` 获取资源
- 返回值通过 `res.json()` 解析为 JSON 对象
- 支持通过 `uiMetadata?.['preferred-frame-size']` 设置宽度

async def resource_proxy(uri: str):
    sse_logger.emit("Frontend", "CF-Agent", "resource-proxy", uri)
    if uri.startswith("ui://stargate/"):
        sse_logger.emit("CF-Agent", "SG-Agent", "MCP resources/read", uri)
        async with httpx.AsyncClient() as client:

### 6. **前端 onReadResource 回调处理流程**
**在 CardMessage.tsx 中的实现：**

```typescript
onReadResource={async ({ uri }) => {
  // 1. 构造请求 URL
  const res = await fetch(`/resource-proxy?uri=${encodeURIComponent(uri)}`);
  
  // 2. 解析 JSON 响应
  return res.json();
}}
```

**完整处理流程：**

1. **Guest UI（iframe）** 需要读取资源时，发送请求：
   ```
   { method: "resources/read", params: { uri: "ui://stargate/card/xxx" } }
   ```

2. **AppRenderer** 调用 `onReadResource` 回调，传入 `{ uri: "ui://stargate/card/xxx" }`

3. **CardMessage** 回调：
   - 发送 HTTP GET 到 `/resource-proxy?uri=ui%3A%2F%2Fstargate%2Fcard%2Fxxx`
   - CF-Agent 接收后转发到 SG-Agent 的 `/mcp/resources/read`
   - 返回 JSON 格式的 ReadResourceResult

4. **AppRenderer** 接收到返回值后：
   - 验证 `contents` 数组
   - 提取 HTML 内容（`text` 字段）
   - 注入到沙箱 iframe 中

## 总结

整个项目形成了一个完整的链路：

```
用户 → CF-Frontend 
      ↓ (/chat)
CF-Agent (resource-proxy)
      ↓ (A2A + /mcp/resources/read)
SG-Agent (返回 mcp_ui_resource)
      ↓ (JSON)

      ↓ (fetch /resource-proxy)

      ↓ (iframe sandbox)
Guest UI (React + MF)
```
