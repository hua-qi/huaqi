[2026-04-13 15:51:13] [user]: 做远端 agent 平台是 AI infra 吗？不是的话，通用名次是什么呢？
[2026-04-13 15:51:20] [assistant]: 做远端 agent 平台可以算 AI infra 的一部分，但更准确的定名取决于侧重点：

- **AI Infra**：偏底层基础设施，如算力调度、模型服务、存储、网络等
- **Agent Platform / Agent Infrastructure**：专指支撑 agent 运行的平台层，更贴切
- **LLMOps**：如果涉及模型运维、编排、监控
- **AI Platform**：更通用，涵盖训练、推理、agent 等

如果你的平台主要负责 agent 的调度、执行、生命周期管理，业界常用 **Agent Platform** 或 **Agentic Infrastructure** 来描述，也有叫 **AI Runtime Platform** 的。
