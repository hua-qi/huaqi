## 4. 日志数据流详细代码结构

### A. 后端日志发射（sse_logger.py 两个模块相同）

```python
# sse_logger.py 核心数据结构
_queues: list[asyncio.Queue] = []  # 所有连接的队列列表

async def subscribe() -> AsyncGenerator[str, None]:
    """SSE 订阅端点：每个连接创建新队列"""
    q = asyncio.Queue()
    _queues.append(q)
    # 发送连接确认事件
    yield f"data: {json.dumps({...})}\n\n"
    # 持续监听和转发队列消息，15秒心跳

def emit(source: str, target: str, msg_type: str, detail: str = ""):
    """日志发射：广播到所有订阅客户端"""
    event = json.dumps({
        "time": _now(),           # 格式: HH:MM:SS
        "source": source,         # 如: "Frontend", "CF-Agent", "LLM"
        "target": target,         # 如: "EventLog", "CF-Agent", "SG-Agent"
        "type": msg_type,         # 如: "chat", "llm-call", "A2A Task"
        "detail": detail          # 可选详情，如消息摘要或参数
    })
    for q in list(_queues):
        q.put_nowait(event)
```

### B. 前端日志订阅（useEventLog.ts）

```typescript
export function useEventLog(urls: string[]) {
  const [entries, setEntries] = useState<EventLogEntry[]>([]);
  const seqRef = useRef(0);  // 全局序号，用于排序

  useEffect(() => {
    const sources = urls.map((url) => {
      const es = new EventSource(url);  // 连接 SSE 源
      es.onmessage = (e) => {
        try {
          const entry: EventLogEntry = JSON.parse(e.data);
          const seq = seqRef.current++;
          const indexed = { ...entry, _seq: seq };
          
          setEntries((prev) => {
            // 保持最后 100 条记录
            const next = [...prev.slice(-99), indexed];
            // 按时间戳排序，相同时间按接收顺序排序
            next.sort((a, b) => {
              const t = a.time.localeCompare(b.time);
              if (t !== 0) return t;
              return a._seq - b._seq;
            });
            return next;
          });
        } catch {}
      };
      return es;
    });
    
    return () => sources.forEach((es) => es.close());
  }, []);

  return entries;  // 返回已排序的日志条目列表
}
```

## 7. 关键技术决策

### ADR-001: MCP-UI Server 职责边界
- MCP-UI Server 只返回 UIResource（HTML 壳子），不预取数据
- 组件在客户端加载后自己取数据，保持 Server 无状态

### ADR-002: UI 组件加载方式 → Module Federation
- 私有组件（`EmployeeChart`）通过 MF 运行时加载
- 与宿主共享 React 实例避免重复加载
- 组件独立部署、独立更新

### ADR-003: 新增 stargate-mcp-ui-server
- 独立 Python 服务作为 stargate-agent 的专属 MCP Server
- 完整实现 MCP-UI 规范（SSE + JSON-RPC）
- tool 和 resource 均在 mcp-server 内定义
