[2026-04-06 22:27:41] [assistant]: ## XP 项目使用说明

XP 是一个 **AI Agent 最佳实践沉淀工具**，让 AI 助手自动积累经验。

### 安装

```bash
python3 -m venv .venv && source .venv/bin/activate
pip install -e .
cp .env.example .env  # 配置 DATABASE_URL 等环境变量
```

### 两种使用方式

**1. MCP 服务器模式**（接入 Cursor / Claude Desktop）

在 MCP 配置文件中添加：
```json
{
  "mcpServers": {
    "xp": {
      "command": "/path/to/.venv/bin/python",
      "args": ["/path/to/xp/src/server.py"]
    }
  }
}
```
Agent 即可使用 `search`（任务前召回经验）、`save`（任务后沉淀经验）、`feedback` 三个工具。

**2. CLI 管理模式**

```bash
xp review        # 审核候选经验
xp stats         # 查看统计数据
xp add           # 手动添加经验
xp promote list  # 查看升层候选
xp analyze       # 质量分析报告
```

### 知识层级
```
Team（团队）→ Business（业务线）→ Project（项目）
```
经验可通过 `xp promote` 命令从项目级逐步升层到团队级。

详细文档见 `QUICK_START.md` 和 `docs/guides/user/cli-guide.md`。
