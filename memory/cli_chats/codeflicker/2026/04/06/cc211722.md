[2026-04-06 16:31:52] [assistant]: ## Supabase 使用教程

Supabase 是一个开源的 Firebase 替代品，提供数据库、认证、存储等功能。

### 1. 安装

```bash
npm install @supabase/supabase-js
```

### 2. 初始化客户端

```js
import { createClient } from '@supabase/supabase-js'

const supabase = createClient(
  'https://your-project.supabase.co',
  'your-anon-key'
)
```

### 3. 数据库操作（CRUD）

```js
// 查询
const { data, error } = await supabase.from('users').select('*')

// 插入
await supabase.from('users').insert({ name: '张三', age: 25 })

// 更新
await supabase.from('users').update({ age: 26 }).eq('id', 1)

// 删除
await supabase.from('users').delete().eq('id', 1)
```

### 4. 用户认证

```js
// 注册
await supabase.auth.signUp({ email: 'user@example.com', password: '123456' })

// 登录
await supabase.auth.signInWithPassword({ email: 'user@example.com', password: '123456' })

// 登出
await supabase.auth.signOut()

// 获取当前用户
const { data: { user } } = await supabase.auth.getUser()
```

### 5. 文件存储

```js
// 上传文件
await supabase.storage.from('avatars').upload('path/to/file.png', file)

// 获取公开 URL
const { data } = supabase.storage.from('avatars').getPublicUrl('path/to/file.png')
```

### 6. 实时订阅

```js
supabase
  .channel('users-changes')
  .on('postgres_changes', { event: '*', schema: 'public', table: 'users' }, payload => {
    console.log('数据变更:', payload)
  })
  .subscribe()
```

### 7. Row Level Security (RLS)

在 Supabase Dashboard 中开启 RLS，并添加策略，例如：

```sql
-- 只允许用户查询自己的数据
CREATE POLICY "用户只能看自己的数据"
ON users FOR SELECT
USING (auth.uid() = user_id);
```

[2026-04-06 16:55:43] [assistant]: **Supabase Edge Functions** 是运行在边缘节点（基于 Deno）的无服务器函数，主要用于：

- **执行服务端逻辑** — 处理不能暴露在前端的敏感操作（如调用第三方 API 并隐藏密钥）
- **Webhook 处理** — 接收 Stripe 支付回调、GitHub 事件等
- **自定义 API 接口** — 构建复杂查询或聚合逻辑，弥补直接查询数据库的不足
- **发送通知** — 触发邮件、短信、Push 通知等
- **数据转换** — 在写入/读取数据库前做预处理

### 简单示例

```ts
// supabase/functions/hello/index.ts
Deno.serve(async (req) => {
  const { name } = await req.json()
  return new Response(JSON.stringify({ message: `Hello ${name}` }), {
    headers: { 'Content-Type': 'application/json' },
  })
})
```

```bash
# 部署
supabase functions deploy hello

# 调用
curl -X POST https://your-project.supabase.co/functions/v1/hello \
  -H "Authorization: Bearer <anon-key>" \
  -d '{"name": "世界"}'
```

核心优势是**靠近用户低延迟**、**可访问环境变量保护密钥**、**与 Supabase Auth/DB 深度集成**。
