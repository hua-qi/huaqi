[2026-04-07 17:22:25] [user]: 现在 xp 可以在本地使用吗？
[2026-04-07 17:25:56] [assistant]: xp 是一个 **MCP Server 工具**，理论上可以本地运行，但有以下前提条件：

**必须满足：**
1. **MySQL 数据库** - 必须有可用的 MySQL 实例（`DATABASE_URL` 必须配置）
2. **LLM API** - 需要 OpenAI 兼容的 API（用于异步提取经验）
3. **配置 `.env`** - 设置 `DATABASE_URL`、`XP_LLM_API_KEY` 等

**启动方式：**
```bash
cp .env.example .env  # 编辑配置
uv sync               # 安装依赖
xp-server             # 启动在 :8000
```
