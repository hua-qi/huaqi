## 关键避坑点

1. **用 `uv` 管理环境**，不要用全局 pip 安装（类比 `nvm` + `package.json`）
2. **类型注解**从一开始就写，前端写 TS 的习惯直接迁移
3. **缩进用空格不用 Tab**，4个空格是规范
4. Python 是**同步为主**，异步（`asyncio`）是进阶内容，先别急着学
5. `==` 比较值，`is` 比较引用（别混用）
