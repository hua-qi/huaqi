[2026-03-31 16:30:18] [assistant]: 以下是使用 Huaqi Growing 制定学习计划的方式：

---

## 通过 CLI 命令

**开始/继续学习某项技能：**
```bash
huaqi study rust          # 首次会自动用 LLM 生成课程大纲
huaqi study "Python 3"
```

**查看所有课程进度：**
```bash
huaqi study --list
```

**重置某个课程：**
```bash
huaqi study rust --reset
```

---

## 通过 Agent 对话

在 `huaqi chat` 中，可以用自然语言触发学习功能：

- "帮我开始学 Go"
- "我 Rust 学到哪了？"
- "给我看一下 Python 的学习计划"
- "完成本章" → 自动推进到下一章

---

## 工作流程

1. `huaqi study <技术名>` → LLM 自动生成 6-10 章大纲
2. 每次只讲一章（约300字）+ 出一道练习题
3. 回答练习题后获得反馈
4. 说「完成本章」标记进度
5. 可启动 `huaqi daemon start`，每晚 21:00 自动推送复习题

---

## 数据存储位置

课程大纲（YAML）和学习会话（Markdown）保存在：
```
~/.huaqi/memory/learning/
├── courses/<slug>/outline.yaml   # 大纲+进度
└── sessions/YYYYMMDD_<slug>.md   # 每次学习记录
```
