### 2.1 Agent 模块 (`huaqi_src/agent/`)

**职责**: LangGraph 架构对话 Agent，支持意图识别、记忆检索、工具调用、人机协同

- `nodes/interrupt_nodes.py` - 人机协同节点

## 🎯 7. 关键工作流

### 对话流程 (Chat Workflow)

```
用户输入
  ↓ HumanMessage
[intent_classifier]      # 分类意图
  ↓
[context_builder]        # 构建系统上下文
  ↓
[memory_retriever]       # 检索记忆
  ↓
[user_analyzer]          # 分析用户理解
  ↓
[chat_response]          # 生成回复（可能调用工具）
  ↓
[tools_condition]        # 需要工具调用？
  ├─ YES → [tools]       # 执行工具
  │          ↓
  │       [chat_response] # 重新思考
  └─ NO  → [extract_user_info]
             ↓
         [interrupt_condition]
         ├─ YES → [require_user_confirmation] # 等待用户确认
         └─ NO  → [save_conversation]
             ↓
            END (AIMessage)
```

### 数据收集流程

```
外部源 (RSS/微信/CLI)
  ↓
[Collectors] → HuaqiDocument
  ↓
[Memory Store]
  ├─ Markdown 存储
  └─ Chroma 向量库
  ↓
[Hybrid Search]  (BM25 70% + Embedding 30%)
```

### 内容流水线

```
RSS/X 源
  ↓ [fetch_sources]
Content 列表
  ↓ [summarizer]
总结内容
  ↓ [translator]
翻译
  ↓ [xiaohongshu_formatter]
小红书格式
  ↓ [publish]
发布到平台
```
