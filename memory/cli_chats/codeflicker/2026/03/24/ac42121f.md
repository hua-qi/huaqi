## 具体含义

Python 虚拟环境（`.venv` 目录）是一个**隔离的 Python 环境**，包含：
- 独立的 Python 解释器
- 独立安装的包（如 `xp`）

**激活前**：必须输入完整路径 `.venv/bin/xp`  
**激活后**：可以直接输入 `xp`

## 如何激活

在 bash/zsh 中运行：

```bash
source .venv/bin/activate
```

激活后，命令行提示符会显示 `(xp)` 前缀，表示已进入虚拟环境：

```
(xp) ➜  xp xp review
```

## 退出虚拟环境

```bash
deactivate
```

```json

      }
    }
  }
}
```

- ✅ 用户只需配置 mcp.json
