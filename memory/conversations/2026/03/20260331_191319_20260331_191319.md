**正确答案：**
```python
b = a[:]  # 这是切片复制，创建了新列表！
print(b)  # → [1, 2]
```

**关键区别在这里：**

```python
# 情况1：直接赋值（引用同一个对象）
a = [1, 2]
b = a        # b 和 a 是同一个对象
a.append(3)
print(b)     # [1, 2, 3]  ← b 也跟着变了

# 情况2：切片复制（创建新对象）
a = [1, 2]
b = a[:]     # 创建了新列表，内容复制过去
a.append(3)
print(b)     # [1, 2]  ← b 不受影响
```

---

### 💡 面试考点

Python 中常见的**浅拷贝**方式：
- `b = a[:]`  ← 切片
- `b = list(a)`  ← 构造函数
- `b = a.copy()`  ← 列表方法

这三者效果一样，都是创建新列表。

## 回合 5

**👤 用户**

可以仅浅拷贝指定索引的列表项吗

**🤖 Huaqi**

可以的！用**切片（slice）**就能做到 👍

```python
a = [1, 2, 3, 4, 5]

# 只拷贝索引 1-3（不包括 4）的元素
b = a[1:4]    # → [2, 3, 4]

# 只拷贝前 2 个
c = a[:2]     # → [1, 2]

# 只拷贝从索引 2 开始到最后的
d = a[2:]     # → [3, 4, 5]

# 只拷贝最后 2 个（负数索引）
e = a[-2:]    # → [4, 5]
```

**注意：** 这仍然是**浅拷贝**——如果列表里嵌套了可变对象，里面的对象还是共享引用：

```python
a = [[1], [2], [3]]
b = a[1:3]      # 只拷贝 [[2], [3]]

b[0].append(99)
print(a)        # [[1], [2, 99], [3]]  ← a 也变了！
```

---

### 💡 实用技巧

如果只需要**单个元素**（不是列表），直接索引就行，不涉及拷贝问题：
```python
item = a[2]  # 只是获取引用，不是拷贝
```

需要深拷贝所有嵌套内容的话，要用 `copy.deepcopy()`。
