**正确答案：`[1, 2, 3]`**

```python
a = [1, 2]
b = a        # b 和 a 指向**同一个**列表对象
b.append(3)  # 修改了这个对象
print(a)     # 所以 a 也变了 → [1, 2, 3]
```

这和 JS 的对象引用是一样的：
```javascript
let a = [1, 2];
let b = a;
b.push(3);
console.log(a); // [1, 2, 3]  ← 一样的道理！
```

**第2题答对啦！** ✅

Python 字符串确实**不可变**（immutable）。不过其实 JS 字符串也是不可变的哦：

```javascript
let s = "hello";
s[0] = "H";  // 不会报错，但也不会生效
console.log(s); // 还是 "hello"
```

只是 JS 允许这种"假装修改"的写法，Python 会直接报错：
```python
s = "hello"
s[0] = "H"  # TypeError: 'str' object does not support item assignment
```
