## 📚 学习计划概览

### 🚀 第1-2周：Python 基础速成
利用你已有的编程思维，快速过一遍Python核心语法：

| 主题 | 重点内容 | 与前端的对比 |
|------|----------|--------------|
| 基础语法 | 变量、数据类型、控制流 | 注意Python的缩进规则和动态类型 |
| 函数 | 定义、参数、返回值、lambda | 比JS更灵活，支持默认参数和关键字参数 |
| 数据结构 | 列表、字典、元组、集合 | 类似JS的Array和Object，但方法更丰富 |
| 文件操作 | 读写文件、上下文管理器(with) | 前端很少接触，重点学习 |
| 异常处理 | try/except/finally | 类似try/catch，但语法更优雅 |

**实践任务**：用Python重写一个你熟悉的JS工具函数（如数据格式化、数组操作）

---

### 🌐 第3-4周：Python Web 框架入门

作为全栈工程师，推荐从 **FastAPI** 或 **Flask** 入手：

**推荐路线**：
- **FastAPI**（推荐⭐）：现代、异步支持好、类型提示友好（对TS用户很亲切）
- **Flask**：轻量、灵活、生态成熟

**学习重点**：
```
✓ 路由定义与请求处理
✓ 请求/响应模型（Request/Response）
✓ 依赖注入系统（FastAPI特色）
✓ 中间件与CORS配置（你前端经验丰富，这个很熟）
✓ 模板引擎（Jinja2）- 理解后端渲染
```

**实践任务**：搭建一个简单的RESTful API，实现CRUD操作

---

### 🗄️ 第5-6周：数据库与ORM

全栈必须掌握数据持久化：

| 技术 | 用途 | 学习要点 |
|------|------|----------|
| PostgreSQL | 关系型数据库 | SQL基础、表设计、索引 |
| SQLAlchemy / Prisma | ORM工具 | 模型定义、迁移、查询 |
| Redis | 缓存 | 基础数据类型、过期策略 |

**实践任务**：设计一个用户系统数据库，包含用户表、文章表，实现关联查询

---

### ⚡ 第7-8周：进阶与工程化

结合前端经验，完善后端工程能力：

1. **认证授权**：JWT、OAuth2（你前端用过，现在学后端实现）
2. **异步编程**：async/await（和JS很像！）
3. **测试**：pytest（单元测试、接口测试）
4. **部署**：Docker、Linux基础、Nginx反向代理

---

### 🏗️ 第9-10周：全栈项目实战

**推荐项目**：个人博客系统 / 任务管理工具

**技术栈示例**：
```
前端：你熟悉的React/Vue
后端：FastAPI + SQLAlchemy
数据库：PostgreSQL
部署：Docker + VPS/阿里云
```

**关键收获**：
- 端到端的数据流设计
- 前后端联调与接口文档（Swagger/OpenAPI）
- 生产环境部署与监控
