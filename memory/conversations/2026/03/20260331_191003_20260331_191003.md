**正确答案：**
```python
b = a[:]  # 这是切片复制，创建了新列表！
print(b)  # → [1, 2]
```

**关键区别在这里：**

```python
# 情况1：直接赋值（引用同一个对象）
a = [1, 2]
b = a        # b 和 a 是同一个对象
a.append(3)
print(b)     # [1, 2, 3]  ← b 也跟着变了

# 情况2：切片复制（创建新对象）
a = [1, 2]
b = a[:]     # 创建了新列表，内容复制过去
a.append(3)
print(b)     # [1, 2]  ← b 不受影响
```

---

### 💡 面试考点

Python 中常见的**浅拷贝**方式：
- `b = a[:]`  ← 切片
- `b = list(a)`  ← 构造函数
- `b = a.copy()`  ← 列表方法

这三者效果一样，都是创建新列表。
